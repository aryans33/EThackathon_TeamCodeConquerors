# 🛰️ ET Radar - Smart Indian Market Intelligence

ET Radar is a professional-grade financial analysis platform designed for Indian equity markets (NSE/BSE). It leverages advanced Large Language Models (LLMs) and automated data pipelines to provide retail investors with actionable signals hidden within corporate filings and technical patterns.

## 🚀 Core Features

- **AI Opportunity Radar**: Automated analysis of BSE/NSE corporate filings using **Groq (Llama 3.3 70B)**. Scores filings for confidence and impact.
- **Technical Pattern Detection**: Native Python implementation for detecting:
  - 📈 52-Week Breakouts
  - ⚔️ Golden & Death Crosses
  - 📉 RSI Oversold Bounces
- **AI RAG Chat**: Context-aware investment assistant that knows your signals, top market movers, and personal portfolio.
- **Automated Data Pipeline**: Background tasks powered by **Celery** to fetch OHLCV prices (yfinance) and BSE filings.

## 🛠️ Tech Stack

- **Backend**: Python 3.10+, FastAPI (Asynchronous)
- **Database**: PostgreSQL (SQLAlchemy Async)
- **AI Engine**: Groq (Llama 3.3 70B & 3.1 8B)
- **Task Queue**: Celery + Redis
- **Data Sources**: yfinance (Prices), BSE (Filings)

## 📋 Prerequisites

- Python 3.10+
- PostgreSQL
- Redis (for Celery & Caching)
- [Groq API Key](https://console.groq.com/keys)

## ⚙️ Setup Instructions

### 1. Installation

```bash
git clone https://github.com/aryans33/EThackathon_TeamCodeConquerors.git
cd et-radar-backend
python -m venv venv
source venv/bin/activate  # venv\Scripts\activate on Windows
pip install -r requirements.txt
```

### 2. Environment Variables (`.env`)

Create a `.env` file in the root directory:

```env
DATABASE_URL=postgresql+asyncpg://user:pass@localhost:5432/etradar
REDIS_URL=redis://localhost:6379/0
GROQ_API_KEY=your_groq_api_key_here
NSE_SYMBOLS=RELIANCE,TCS,INFY,HDFCBANK,ICICIBANK
FRONTEND_URL=http://localhost:3000
```

### 3. Database Initialization

```bash
# Apply migrations or run initial setup
alembic upgrade head
# Seed initial data (Prices, Filings, AI Analysis)
python scripts/seed_data.py
```

## 🏃 Running the Application

### Start the API Server

```bash
uvicorn app.main:app --reload
```

### Start the Background Worker (In a new terminal)

```bash
celery -A app.tasks.celery_app worker --loglevel=info -P eventlet
```

### Test the AI Chat

```bash
python scripts/test_chat.py
```

## 📡 API Endpoints

- `GET /api/stocks`: List tracked stocks
- `GET /api/signals`: Fetch AI-generated market signals
- `POST /api/chat`: Stream AI responses with RAG context
- `GET /api/status`: Check system health & tracker stats
